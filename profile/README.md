## Overview
Open-Bodycam is a personal research project exploring
- Wearable camera system architecture
- Embedded video pipelines using libcamera
- Low-power edge devices
- Real-time frame processing
- Hardware prototyping with 3D printing

The current prototype is built around a Raspberry Pi Zero 2W and demonstrates stable 1080p recording, custom overlay rendering, and a compact body-mounted form factor. With more feature and eco-system on the horizon.

### Features
- 1080p video recording
- Real-time timestamp and logo overlays
- Wearable 3D-printed enclosure
- Low-power embedded architecture
- Fully open software stack

### Hardware
Current prototype hardware
- Raspberry Pi Zero 2W
- Raspberry Pi camera module 3 Wide
- Custom 3D-printed enclosure
- Battery power system (18650 + TP4056)
